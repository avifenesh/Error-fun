Error Fortune Cookies - Examples

This page demonstrates different ways to use the Error Fortune Cookies library.

Basic Usage

Generate a fortune with JavaScript:

ErrorFortune.crack("TypeError: Cannot read property 'length' of undefined", {
  target: "basic-example",
  style: "confucius"
});

Click the button to generate a fortune

Custom Element

Use the custom element for easy embedding:

<error-fortune 
  message="ReferenceError: x is not defined" 
  style="haiku"
  theme="dark">
</error-fortune>

Error Handling Integration

Integrate with try/catch blocks:

try {
  // This will cause an error
  const obj = null;
  document.getElementById("result").textContent = obj.nonExistentProperty;
} catch (error) {
  ErrorFortune.crack(error.toString(), {
    target: "error-example",
    style: "blame"
  });
}

Click the button to trigger an error

Shareable Links

Generate shareable links with URL parameters:

function createShareableLink(error, style, theme) {
  const url = new URL(window.location.href);
  url.searchParams.set('error', encodeURIComponent(error));
  url.searchParams.set('style', style);
  url.searchParams.set('theme', theme);
  return url.toString();
}
