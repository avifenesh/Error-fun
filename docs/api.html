Error Message Fortune Cookies API Documentation

Overview

Error Message Fortune Cookies provides a simple JavaScript API for transforming error messages into fortune cookie wisdom. This document outlines the available methods, options, and integration approaches.

Installation

NPM

npm install error-fortune-cookies

CDN

<script src="https://cdn.jsdelivr.net/npm/error-fortune-cookies@latest/dist/error-fortune.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/error-fortune-cookies@latest/dist/error-fortune.min.css">

Core API

ErrorFortune.init(options)

Initializes the Error Fortune Cookies library with custom options.

ErrorFortune.init({
  animation: true,
  defaultStyle: 'confucius',
  defaultTheme: 'zen'
});

Options

Option Type Default Description
animation Boolean true Enable/disable animations
defaultStyle String 'confucius' Default fortune style
defaultTheme String 'zen' Default visual theme

ErrorFortune.crack(errorMessage, options)

Transforms an error message into fortune cookie wisdom.

ErrorFortune.crack("TypeError: Cannot read property 'length' of undefined", {
  style: "haiku",
  theme: "dark",
  target: "fortune-container",
  animation: true,
  callback: function(fortune) {
    console.log("Fortune generated:", fortune);
  }
});

Parameters

Parameter Type Required Description
errorMessage String Yes The error message to transform
options Object No Configuration options

Options

Option Type Default Description
style String 'confucius' Fortune style to use
theme String 'zen' Visual theme
animation Boolean true Enable/disable animations
target String/Element null Target element to display the fortune
callback Function null Callback function called after fortune generation

Available Styles

  • confucius - Ancient wisdom about your coding practices
  • haiku - Poetic verses about your technical troubles
  • tarot - Mystical readings about your code's future
  • blame - Creative excuses for why your code failed
  • motivational - Inspirational posters from your failures

Available Themes

  • zen - Calm, peaceful design
  • chaos - Energetic, chaotic design
  • dark - Dark mode
  • light - Light mode

Return Value

The crack() method returns a fortune object with the following properties:

{
  original: "TypeError: Cannot read property 'length' of undefined",
  wisdom: "Confucius say: Man who measure nothing find himself coming up short.",
  style: "confucius",
  timestamp: "2025-08-18T12:34:56.789Z"
}

Custom Elements

Error Fortune Cookies provides a custom element for easy embedding in HTML.

<error-fortune 
  message="ReferenceError: x is not defined" 
  style="haiku"
  theme="dark"
  animation="true">
</error-fortune>

Attributes

Attribute Type Default Description
message String Required The error message to transform
style String 'confucius' Fortune style to use
theme String 'zen' Visual theme
animation String 'true' Enable/disable animations

URL Parameters

You can generate fortunes directly from URL parameters:

https://your-site.com/error-fortune/?error=TypeError%3A%20undefined%20is%20not%20a%20function&style=haiku&theme=dark

Available Parameters

Parameter Description
error URL-encoded error message
style Fortune style
theme Visual theme

Advanced Usage

Adding Custom Fortune Styles

You can add your own fortune styles by extending the transformers object:

ErrorFortune.transformers.myCustomStyle = function(error) {
  // Transform the error message
  return "My custom wisdom about: " + error;
};

// Use your custom style
ErrorFortune.crack("Some error", { style: "myCustomStyle" });

Integration with Error Handling

try {
  // Your code that might throw an error
} catch (error) {
  ErrorFortune.crack(error.toString(), {
    target: "error-container",
    style: "confucius"
  });
}

Creating Shareable Links

function createShareableLink(error, style, theme) {
  const url = new URL(window.location.href);
  url.searchParams.set('error', encodeURIComponent(error));
  url.searchParams.set('style', style);
  url.searchParams.set('theme', theme);
  return url.toString();
}

Browser Compatibility

Error Message Fortune Cookies is compatible with all modern browsers:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)

Internet Explorer is not supported.
